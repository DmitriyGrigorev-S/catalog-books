# Каталог книг

Адаптивная вёрстка блока каталога книг по макету с использованием методологии БЭМ, препроцессора SCSS и чистого JavaScript без фреймворков.

Реализована загрузка данных из локального JSON, логика распределения карточек по слотам и перемешивание элементов в соответствии с условиями технического задания.

## Стек технологий
- **HTML5**
- **SCSS (SASS)** — компонентный подход
- **JavaScript (Vanilla JS)** — без фреймворков
- **BEM** — семантичные, масштабируемые имена классов
- **Fetch API** — загрузка данных

## Общий подход
- HTML содержит **каркас сетки** (grid и слоты)
- Все данные карточек (обложка, название, автор, описание, сортировка) хранятся в **локальном JSON**
- Контент **рендерится на клиенте** после загрузки данных
- Размер карточки определяется **местом вставки в сетку**, а не данными из JSON

## Логика работы

### **Загрузка данных**
Данные книг загружаются из `books.json` при помощи `fetch`
После успешной загрузки выполняется первичный рендер каталога

### **Распределение карточек**
В HTML определены слоты с атрибутами `data-slot`
В зависимости от слота карточке автоматически назначается размер:
- большая — без модификатора
- средняя — `card-product--medium`
- маленькая — `card-product--small`


### **Перемешивание (shuffle)**
Книги без параметра `sort` участвуют в случайном перемешивании
Книги с одинаковым `sort`:
- фиксируются в одной зоне
- перемешиваются только между собой

Книги с разным `sort` не влияют друг на друга
Повторное перемешивание выполняется по кнопке без повторной загрузки данных


## **Адаптивная вёрстка**
- Контейнер шириной **1140px** для desktop
- На мобильных устройствах используется горизонтальный скролл
- Сетка адаптируется под разные разрешения
